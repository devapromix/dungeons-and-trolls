Commands:
help - Show this help message.                                  north (or n) - Move player north.
about - Show information about game.                            south (or s) - Move player south.
new - Starts a new game.                                        east (or e) - Move player east.
save - Saves a game.                                            west (or w) - Move player west.
load - Loads a saved game.                                      up (or u) - Move up.
map - Display the game map.                                     down (or d) - Move down.
status - Display player's stats.
skills - Display player's skills.
rest - Rest to restore health and mana.
time - Displays the current time.
light - Light a fire.
look - Displays the current location.
items - Display player's inventory and gold.
eat <item> - Eats an item.
drink <item> - Drinks an item.
pick [quantity] <item> - Pick up an item.
drop [quantity] <item> - Drop an item.
attack <enemy> - Attack an enemy to engage in combat.
volume <level> - Set the audio volume (0 to 10).
quit - Exit the game.
